# Eden Core

Proyecto base para las pruebas de Edén. Incluye login funcional y soporte para i18n.

## Scripts útiles

- `npm install`: instala las dependencias.
- `npm test`: ejecuta las pruebas (aún sin implementar).
- `npm run dev`: inicia el servidor de desarrollo.
- `npm run build`: genera la versión de producción.

## Mandamientos

Consulta `src/constants/mandamientos.ts` para ver los mandamientos del Edén.
